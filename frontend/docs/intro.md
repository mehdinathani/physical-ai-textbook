# Physical AI & Humanoid Robotics Textbook

Welcome to the comprehensive textbook on Physical AI and Humanoid Robotics. This resource covers the complete technology stack from foundational concepts to advanced implementation.

## Course Structure

This textbook is organized into five comprehensive modules:

### Module 0: Foundations
- Core concepts of Physical AI
- Hardware platforms (Jetson & RTX)
- Lab architecture and safety
- Humanoid robotics landscape

### Module 1: ROS 2 (Nervous System)
- ROS 2 architecture fundamentals
- Python agents with rclpy
- Robot description with URDF
- Launch systems and deployment

### Module 2: Digital Twin (Simulation)
- Gazebo physics simulation
- World building with SDF
- Unity visualization
- Sensor simulation

### Module 3: Isaac AI (The Brain)
- Isaac Sim introduction
- Visual SLAM algorithms
- Nav2 path planning
- Sim-to-real transfer techniques

### Module 4: VLA (Vision-Language-Action)
- Voice-to-action with Whisper
- LLM cognitive planning
- Humanoid locomotion
- Capstone pipeline integration

## Getting Started

Navigate through the modules using the sidebar to begin your journey into Physical AI and Humanoid Robotics. Each module builds upon previous knowledge with practical examples and implementation guidance.
